✅ Application Status

This frontend is successfully deployed and connected to the backend API via the Apache HTTP proxy.

📡 Available API Endpoints

GET /api/persons

Retrieve all persons from the database

curl http://your-server/api/persons

GET /api/persons/{id}

Retrieve a specific person by ID

curl http://your-server/api/persons/1

POST /api/persons

Create a new person

curl -X POST http://your-server/api/persons -H "Content-Type: application/json" -d '{"name":"John","email":"john@example.com"}'

🏗️ Architecture

Deployment Stack

This application is deployed using:

  • Database: PostgreSQL 14.1
  • API: Spring Boot (Java 21)
  • Proxy: Apache HTTP Server 2.4
  • Frontend: Static HTML/CSS
  • Orchestration: Docker & Ansible
  • CI/CD: GitHub Actions
